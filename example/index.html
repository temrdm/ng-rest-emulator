ng-rest-emulator

Angular module for change preset using rest-emulator and gulp plugin.

Open DevTools console, and set mock preset using restEmulator.set(). Then click "Update" button. For example: 

                restEmulator.set('/api/v1/users', 'default');

Available methods: 

                restEmulator.set(url, preset, [method]);

                restEmulator.get(url, [method]);

                restEmulator.clear(url, [method]);

                restEmulator.list();
